# Startup-Funding-Survival-Analysis
This project explores the dynamics of startup funding in India, with a focus on understanding what drives startup survival and growth. Using a real-world funding dataset, I performed end-to-end analysis covering data cleaning, exploratory analysis, SQL-based insights, statistical testing, and machine learning.

The workflow began with preprocessing messy financial and categorical data, followed by feature engineering to derive meaningful indicators such as funding tiers, deal size categories, and city-level segmentation. SQL was then used to uncover patterns in industry risk, investor activity, funding concentration, and regional ecosystem strength.

A key part of the analysis involved identifying survival trends by examining whether startups were able to secure multiple funding rounds. Statistical methods, including hypothesis testing, were applied to validate differences across cities and funding stages. Additionally, machine learning models such as Logistic Regression, Decision Trees, and Random Forest were built to predict startup survival, with performance evaluated using ROC-AUC metrics.

The findings highlight strong relationships between funding size, location, and survival probability, while also revealing the unequal distribution of capital across startups. Visualizations were created to clearly communicate trends in deal flow, industry risk, and funding concentration.

Overall, this project demonstrates a practical application of data analytics in understanding real-world business ecosystems, combining technical rigor with actionable insights for founders, investors, and policymakers.
